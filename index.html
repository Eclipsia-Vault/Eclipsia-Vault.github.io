How to get a free TMDB API key
1 Create a free account at themoviedb.org.
2 Once logged in, go to Settings → API, then click Create under "Request an API Key".
3 Choose Developer, accept the terms, and fill in the short application form. For personal use, "Application Name" and "Application URL" can be anything (e.g. Nuvio and a placeholder URL).
4 Once approved (usually instant), copy the API Key (v3 auth) value from your new API settings page.
5 Paste it into Nuvio → Settings → Integrations → TMDB Enrichment and save.
